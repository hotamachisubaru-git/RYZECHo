# RYZECHØ Prototype v0.0.3

公開日: 2026-03-27

`v0.0.3` は、`v0.0.2` 系で見つかった起動クラッシュと保存データまわりの落ちやすさを潰す安定化リリースです。ゲームループやコンテンツ量は維持しつつ、ローカル実行で即終了しないことを最優先に調整しました。

## この版で修正・改善したもの

- 投資状態初期化の相互再帰を解消し、起動直後の `StackOverflowException` を修正
- チーム総投資額の集計を辞書直接参照へ変更し、初期化処理の再入を防止
- 保存データ署名が無効な場合でも即終了せず、署名情報を再生成して継続できるフェイルセーフを追加
- `README`、アプリ表示名、バージョン情報を `v0.0.3` に更新

## この版で維持している前提

- WinForms ベースのトップダウン戦術スライスで、完成版 3D FPS そのものではない
- エージェント 6 名の固有スキルと ULT 発動本体は Unity 側で実装する前提
- プロフィール保存と進行管理はローカル JSON ベース
- 広告表現はプレイを止めない外周サイン型に限定

## 既知の制限

### エージェント

- 固定キャストのみで、ヴェール、ヴァイン、ニトロ、オアシス、ディバイド、グリッチのスキルは未実装
- ULT ポイントはリソース進行までで、各エージェントの ULT 消費スキルは未実装

### ルールとマップ

- マップは固定の A/B 2 サイト構成で、可変マッププールや本番用 UI は未実装
- 構築レギュレーションは近似的な 3 レーン審査で、完全な検証ツールではない

### 継続性

- オンライン同期、サーバー権威の進行、課金導線、本番用 inventory は未実装
