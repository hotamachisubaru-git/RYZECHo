# RYZECHØ Prototype v0.0.2

公開日: 2026-03-21

`v0.0.2` は、`v0.0.1` で未反映だったコア仕様のうち、WinForms プロトタイプで成立させられる範囲をまとめて反映した更新です。A/B 2 サイト、4 人個別投資、ULT ポイント、サブウェポン、シェア・ビジョン、一覧型ショップ UI を追加し、タクティカルスライスとしての密度を引き上げました。

## この版で追加・更新されたもの

- A/B 2 サイト構成のボムルールと、サイト状態が分かる HUD / ミニマップ表示
- 4 人個別投資 UI と、チーム総投資額へ同期するボス経済
- 敵ボス撃破 `ULT +2` を含む ULT ポイント進行
- メイン武器 9 種 + サブウェポン 2 種のロードアウト、戦闘中の持ち替え
- ベットフェーズの一覧型ショップ UI
- 味方共有のシェア・ビジョンと、画面端インジケーターを含む音情報共有
- リコンビーコン、シールドリレーを含む設置物追加
- A/B 2 サイト向けの 3 レーン維持チェックを含む構築レギュレーション

## この版で維持している前提

- WinForms ベースのトップダウン戦術スライスで、完成版 3D FPS そのものではない
- エージェント 6 名の固有スキルと ULT 発動本体は Unity 側で実装する前提
- プロフィール保存と進行管理はローカル JSON ベース
- 広告表現はプレイを止めない外周サイン型に限定

## 既知の制限

### エージェント

- 固定キャストのみで、ヴェール、ヴァイン、ニトロ、オアシス、ディバイド、グリッチのスキルは未実装
- ULT ポイントはリソース進行までで、各エージェントの ULT 消費スキルは未実装

### ルールとマップ

- マップは固定の A/B 2 サイト構成で、可変マッププールや本番用 UI は未実装
- 構築レギュレーションは近似的な 3 レーン審査で、完全な検証ツールではない

### 継続性

- オンライン同期、サーバー権威の進行、課金導線、本番用 inventory は未実装
