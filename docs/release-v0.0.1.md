# RYZECHØ Prototype v0.0.1

公開日: 2026-03-21

`v0.0.1` は、WinForms ベースでコア戦術ループを検証するための最初の公開版です。完成版ではなく、1 サイト構成のタクティカルスライスを安定して配布できる状態に整えたリリースとして扱います。

## この版で入っているもの

- 4v4、7 先、6-6 以降 OT のラウンド進行
- 第 4 ラウンド後の攻守交代と再エディット
- 1 サイトのボム設置、解除、爆破タイマー
- 9 種のメイン武器テーブルと簡易ロードアウト
- ボス選択、簡易投資、撃破配当を含むボス経済のコアループ
- 120 度視界、`SR` 100 度視界、足音と銃声の簡易可視化
- 初回構築、簡易レギュレーション、破壊可能オブジェクト
- ローカル保存のプロフィール、XP、ランク、契約進行
- 外周サイン型の広告テーマ切替
- ランタイム整合性チェックと保存データ改ざん検知

## 既知の制限

### ボス経済

- 4 人個別投資 UI は未実装で、WinForms 版では `_selectedBet` をチーム総投資額として近似
- ULT ポイント系そのものは未実装で、敵ボス撃破時の `ULT +2` は未反映

### 武器

- サブウェポン 2 種の個別選択、購入、持ち替えは未実装
- 武器選択 UI は `Q / E` 切替ベースの簡略版で、ショップ一覧 UI までは未反映

### ルール

- 1 サイト構成のタクティカルスライスで、マルチサイトや本番向けの攻守 UI までは未反映

### 視界と情報

- シェア・ビジョン未実装
- 音イベントは床面矢印ベースの簡易版で、画面端演出や味方共有との整合は未調整

### マップエディット

- 設置物の種類が限定的
- レギュレーションは現行 1 サイトマップ向けの簡易ヒューリスティックで、完全な 3 レーン審査までは未実装
