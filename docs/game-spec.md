# RYZECHØ | Game Design Document (GDD)

**Project Status:** Production Blueprint  
**Last Updated:** 2026-05-30  
**Target Platform:** PC (Competitive FPS)

---

## 1. Executive Summary

`RYZECHØ` は、**「100度の視界」「260度の死角」「音の可視化」「シェア・ビジョン」「動的マップエディット」**を核とした、次世代 4v4 タクティカルシューターです。

従来の FPS が「いかに早く敵を見つけるか」という反射神経のゲームであるのに対し、本作は**「見えない場所をどう管理するか」というリスクマネジメントのゲーム**へと転換します。不完全な視界の中で情報を精査し、リスクを取って資産を増やす「投資型サバイバル」です。

---

## 2. Five Core Differentiators (5つの決定的差別化ポイント)

### 2.1 「100度の視界」と「260度の死角」
- プレイヤーの視野を意図的に 100 度に制限。残りの 260 度は「濃いもや（フォグ）」で覆う。
- 従来の FPS が「いかに早く敵を見つけるか」を競うのに対し、本作は「見えない場所をどう管理するか」を競う。

### 2.2 シェア・ビジョン（視界共有）システム
- 味方5人の視界（100度×5）をミニマップ上に統合。
- チームでフォーメーションを組むことで「360度の安全」を確保。「孤立＝死」を物理的に定義し、究極のチームプレイを強制・促進する。

### 2.3 音の視覚化（0.3秒の波紋と矢印）
- 銃声（赤）、スキル（黄）、足音（白）を、発生源から 0.3 秒で消える波紋と矢印で表示。
- 足音が「3歩に1回」しか出ない仕様により、情報の隙間を突く「音の心理戦」を確立。

### 2.4 プレイヤー主導の「動的マップ・エディット」
- ベースマップは不変だが、プレイヤーが設置する扉や罠は破壊可能。
- 毎ラウンド、戦場の形が書き換わるため、攻略のパターン化を許さない。

### 2.5 インゲーム・ダイナミック・アド（没入型収益）
- 動画広告ではなく、ゲーム内のビルボードや設置物に実在の広告を表示。
- プレイ体験を中断させず、サイバーパンクな世界観を補強する装置として広告を機能させる。eスポーツ放送自体が巨大な広告媒体となるビジネスモデル。

## 3. Competitive Philosophy (競合との違い)

| 要素 | 従来のFPS (VALORANT等) | 当ゲーム | プレイヤーの得られる体験 |
|---|---|---|---|
| 視覚情報 | 全方位（広視野角） | 120度限定 ＋ 音の可視化 | 死角を「推理」する知的なスリル |
| マップ | 固定（テンプレ化） | 初期エディットで変化 | 常に新しい戦略を考える創造性 |
| 経済 | 作業的な売買 | ボスへの投資・配当 | チーム全員で大博打を打つ一体感 |
| 勝利条件 | 撃ち合いの強さ | 情報網の構築 ＋ 経済管理 | 脳と反射神経の両方を使う達成感 |

### 3.1 既存FPSへの反逆と革新

**1. 「視覚の暴力」から「聴覚の想像力」へのシフト**
- 多くのFPSは「いかに早く敵を見つけるか」という視覚の反射神経ゲームになっている。180度に近い広い視野角と高画質なグラフィックは情報を「与えすぎ」ており、プレイヤーの想像力を奪っている。
- 本作は視界を 100 度（人間が集中できる限界）に絞り、残りの 260 度を音の波紋（オーディオ・リップル）で補完させる。「見えたものを撃つ」受動的なゲームから、「聞こえる音から敵の姿を脳内に構築する」能動的なゲームへの進化。

**2. 「定石」を破壊するマップエディット**
- VALORANTやCS2のようなタクティカルFPSは、発売から数ヶ月で「最強の守り方」「テンプレの攻め方」が固定され、プレイヤーは「覚えゲー」を強いられる。
- 最初のラウンド前にプレイヤー自身が戦場を書き換えられる。強化扉やハチミツトラップの配置によって「毎試合、異なる構造のマップ」が誕生する。

**3. 「作業的な経済」を「魂のギャンブル」へ**
- 「勝ったから買う、負けたから貯める（エコ）」という既存のマネーシステムは作業化している。逆転のチャンスは少なく、経済格差がそのまま敗北に直結するストレスがある。
- 「バウンティ・ボス」への投資システムにより、たとえ劣勢でもボスが生き残れば一気に富を得て逆転できる。「一発逆転のギャンブル性」がエンターテインメント性を生む。

## 6. Bounty Economy (ボスシステム)

### 6.1 選出と投資（バウンティ・セレクション）

- **選出タイミング:**
  - 初回（第1R前）：60秒の準備フェーズ後半
  - 第2R以降：45秒の準備フェーズの最初
- **回数制限:** 1試合（OT含む）を通じ、同じプレイヤーがボスになれるのは最大2回まで
- **選出人数:** 各チームから必ず1名を選出
- **選出プロセス:** 立候補・推薦フェーズ → 重複時は多数決またはシステム自動割り当て（回数が残っているプレイヤーから優先）

### 6.2 投資とバフ（負の二次関数）

- 味方全員が自分の所持金からボスへ自由に投資できる。
- **合計300円投資が最もステータス上昇率（HP・速度）が高くなる「負の二次関数」モデル**。300円を超えると上昇幅は急激に鈍化する。
- 外見的変化：ボスは専用のエフェクト（背中のデバイスが発光する、頭上に紋章が出る）を纏う。

### 6.3 リスクとリターン

| シナリオ | 結果 |
| :--- | :--- |
| **ボス・キルボーナス** | ボスが敵を1人倒すごと、生き残っている味方全員に **200円** が即座に支払われる |
| **配当（成功）** | ボスが「キル1以上＋生存勝利」の場合、味方それぞれが投資額の **2倍** を返還される |
| **ボス死亡（失敗）** | 投資した金はすべて失われる。敵チームに **800円** と **ULT +2** を付与 |

### 6.4 戦略的運用

- **「投資リレー」戦略:** お金を持っているプレイヤーが腕の良いプレイヤー（ボス）に投資し、ボスが生き残ることでチーム全体の資産を倍増させる「資産運用」的な立ち回り。
- ボスは単なる「強化された兵士」ではなく、チームの「動く投資先」となる。

## 8. Tactical Construction (マップエディット)

### 8.1 基本ルール

- **発生タイミング:** 第1ラウンド、攻守交代後直後のラウンド開始前のみ。試合開始前の長い準備時間（60秒〜90秒）に一度だけ行われる。
- **設置物の維持:** ここで設置したものは、全ラウンドを通して維持される（破壊された場合を除く）。ラウンド間のインターバルでは設置の修正や追加はできない。
- **建築ポイント（AP）制:** チーム共有ポイントで、武器を買う「所持金」とは完全に別枠。

### 8.2 設置オブジェクト・カタログ

#### 妨害・トラップ系（敵の足を止め、隙を作る）

| オブジェクト | 効果 | 戦略 |
| :--- | :--- | :--- |
| **ハチミツ・パッチ** | 床に粘着性の合成蜜。踏んだ敵は移動速度50%低下＋特大の音発生 | 死角に設置し、音だけで敵の位置を確定 |
| **害虫の繭** | 壁に設置するバイオ兵器。敵が接近すると破裂し、追尾羽虫が視界をバグらせる | ヴェールの「もや」と組み合わせ、敵を完全な暗闇に |
| **トリップワイヤー・ガス** | ワイヤー設置。切った瞬間に微量の毒ガスを放出し、敵を咳き込ませる | 咳の音は遮蔽物を突き抜けて響くため、壁抜きショットの目印に |

#### 構造変化・防御系（ルートを制御する）

| オブジェクト | 効果 | 戦略 |
| :--- | :--- | :--- |
| **強化ナノ・ゲート** | 厚みのある白い金属扉。通常の射撃ではびくともしない。一定以上のダメージが必要 | 最短ルートを封鎖し、敵をトラップだらけの遠回りルートへ誘導 |
| **一方向バイザー壁** | 設置した側からは透明、反対側からは壁に見える。通り抜けは1発撃たないと不可 | 死角からの奇襲・迂回ルートの確保 |
| **ポータブル・カバー** | 何もない通路に設置できる、胸の高さまでの遮蔽物 | 敵の予想外の場所に「射線を通せる遮蔽物」を作り定石を崩す |

#### 情報・撹乱系（音と視覚で騙す）

| オブジェクト | 効果 | 戦略 |
| :--- | :--- | :--- |
| **反響スピーカー** | 録音された「足音」や「リロード音」をランダムに再生し続ける | 260度の死角に音の波紋を出し続け、敵のエイムを誤った方向へ |
| **ホログラム・デコイ** | キャラ立ち姿を投影するホログラム。攻撃を受けると消滅 | 角に設置して敵に弾を無駄撃ちさせると同時に、その位置を音で把握 |

**すべての構造物は壊せるようにし、壊されると次のラウンドから現れない。**

### 8.3 設計制限（Anti-Abuse）

- **ノー・ビルド・ゾーン:** リスポーン地点出口から半径数メートル以内、目標物直上・重なる位置は設置不可。
- **メインルートの確保:** 3つの主要ルートのうち、最低でも1つのルートは「歩いて通れる隙間」が常に確保される。全ルートを完全封鎖することは不可。
- **デッドゾーン設定:** トラップを設置した際、その周囲一定範囲には同じカテゴリーの設置物を置けない。「理不尽なキルルーム」の作成を防止。
- **連結制限:** 強化扉などの大型設置物は、横に並べて設置できる数に上限を設ける。

## 9. Audio Ripple (音響視覚化詳細)

### 9.1 足音の波紋

- **発生頻度:** 3歩に1回のペース
- **形状:** 白く薄い正円
- **持続時間:** 発生から0.3秒で素早く消滅
- **効果:** 敵が「歩いている」のか「走っている」のかは、波紋が出る「テンポ」で判断。連続して出ないため、フェイントが可能。

### 9.2 銃声とスキルの矢印表示

- **発生条件:** 死角（260度）で音が鳴った瞬間に、画面端または床面に「矢印」を表示
- **銃声（赤色）:** 鋭い赤い矢印で、撃ってきた方向を指し示す
- **スキル（黄色）:** ギザギザした黄色の矢印で、発動場所を指し示す
- **持続時間:** どちらも0.3秒で消滅。撃ち続けている間はチカチカと点滅

### 9.3 壁越しの減衰

- 壁を一枚挟むと、矢印や波紋の「不透明度」が下がり、輪郭がぼやける。
- 「目の前の壁の裏」なのか「遠くの遮蔽物の裏」なのか、視覚情報だけでは断定させず、音の大きさ（聴覚）との合わせ技で判断させる。

### 9.4 アンチ・キャンプ（居座り対策）

- 10秒以上「移動」も「スキル発動」もせず一箇所に留まっている場合、エージェントの呼吸音が大きくなる。
- この音は非常に薄くぼやけた「円状の波紋」として、超至近距離（壁一枚挟んだ隣など）の敵にのみ表示される。
- 擦れる音などの余計なノイズは排除し、「生命維持による最小限の音」のみをペナルティとする。

### 9.5 環境音と伝播

- **材質による音色の変化:** 地面の材質（コンクリート、鉄板、ハチミツ）により、聞こえる足音の「音色」のみが変化する。
- **遮蔽物の減衰:** 壁を一枚挟むと、音の波紋は「遠くの音」と同様にぼやけて薄くなる。

## 11. Competitive Philosophy (競合との位置付け)

> 当ゲームは、単に「撃ち合う」ゲームではありません。**「不完全な視界の中で、いかに情報を精査し、リスクを取って資産を増やすか」を競う、極めて現代的な「投資型サバイバル」**です。
>
> 既存のFPSが「エイム（物理的技術）」に寄りすぎている今、このゲームは「判断（知的技術）」に重きを置くことで、新しい競技シーンのスタンダードを構築できる可能性を秘めています。

## 12. Business Model (収益化・ビジネスモデル)

### 12.1 基本理念：公平性の担保

- **完全脱Pay to Win:** 課金によってキャラクターの性能、武器の威力、設置物の耐久度が上がることは一切ない。
- **競技性の維持:** 全ての課金要素は「見た目（ビジュアル）」と「自己表現」に特化する。

### 12.2 収益化要素

| カテゴリ | 内容 |
| :--- | :--- |
| **アーキテクト・カスタマイズ** | 設置物の外見スキン（強化扉を「和風の襖」「ハイテクなレーザーゲート」等に変更）、トラップのエフェクト、建築ホログラム |
| **インゲーム・ダイナミック・アド** | サイバー・ビルボード（ビル壁面や巨大モニターに実在企業のロゴをホログラム表示）、設置物アド・パネル、大会連動広告 |
| **視聴覚・UIカスタマイズ** | バイザー・フレーム（死角との境界線を飾るHUDスキン）、波紋フィニッシャー（最後のキル演出のカスタマイズ）、キル・アンセム（ボス選出・勝利時のBGM） |
| **シーズンパス ＆ エージェント解放** | シーズンごとに限定の武器スキンや設置物スキンを報酬として配置。エージェント解放はプレイで貯まる通貨でも可能、課金による即時解放も提供 |
