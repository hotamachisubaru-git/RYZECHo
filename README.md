# RYZECHØ

`RYZECHØ` は、120 度の限定視界、音の可視化、初期マップエディット、ボス投資経済を核にした 4v4 タクティカルシューターのプロジェクトです。  
このリポジトリには、WinForms 製のプレイアブルプロトタイプと、今後の実装基準になる仕様書を置いています。

現在の公開版は `RYZECHo.Prototype v0.0.1` です。完成版 FPS ではなく、1 サイト構成のタクティカルスライスを先に触れるための Windows 向けプロトタイプとして扱います。

## 現在の実装物

現状の `RYZECHo.Prototype` は、フル 3D の完成版ではなく、コア体験を素早く触るためのトップダウン戦術スライスです。主に次の要素を先行検証しています。

- 初回構築フェーズ
- ラウンド前の 4 人ボス選択と簡易投資
- 4v4、1 サイト、ボム設置と解除、攻守交代、OT を含むラウンド進行
- 120 度視界と `SR` 100 度視界を前提にした戦闘
- 波紋 UI と死角矢印による簡易的な音の可視化
- ローカル保存のランク、XP、契約進行、見た目切替の基礎

未反映項目は [プロトタイプ差分メモ](docs/prototype-alignment.md) に整理しています。

## 技術構成

- `.NET 10`
- `Windows Forms`
- C# と GDI+ による描画

## 起動

```powershell
dotnet run --project .\RYZECHo.Prototype\RYZECHo.Prototype.csproj
```

## 操作

### 構築

- `1 / 2 / 3`: 建築物選択
- `Tab`: 建築物を順送り
- `左クリック`: 強調スロットに設置
- `右クリック`: 設置物を撤去して AP を返還
- `Q / E`: 設置物スキン切替
- `R`: 会場広告テーマ切替
- `Enter`: 構築確定

### ラウンド準備

- `1 / 2 / 3 / 4`: ボス選択
- `Q / E`: 武器変更
- `A / D`: 投資額調整
- `Enter`: 出撃

### 戦闘

- `W / A / S / D`: 移動
- `マウス`: 照準
- `左クリック長押し`: 射撃
- `F`: ボム設置 / 解除

### 共通

- `Space`: 説明オーバーレイ表示切替
- `R`: 勝敗後に再挑戦

## 既知の制限

- 1 サイト構成の WinForms プロトタイプで、マルチサイトや本番向け UI は未実装です
- エージェント 6 名の固有スキルと ULT ポイント進行は未実装です
- ボス経済は 4 人個別投資ではなく、チーム総投資額 `_selectedBet` による近似です
- 武器ショップは簡略版で、サブウェポン選択や一覧 UI は未実装です
- 視界共有と音情報の味方共有は未実装です

詳細は [v0.0.1 リリースノート](docs/release-v0.0.1.md) と [プロトタイプ差分メモ](docs/prototype-alignment.md) を参照してください。

## Assets

- `RYZECHo.Prototype/Assets` 配下に `Audio / Characters / Environment / Weapons / UI / VFX / Fonts / Data` を整理済みです
- `tools/Generate-PlaceholderAssets.ps1` で仮アセットを再生成できます
- 本番アセットは同名差し替えか、各カテゴリへの追加で運用できます
