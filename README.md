# RYZECHØ

`RYZECHØ` は、100 度の限定視界、音の可視化、初期マップエディット、ボス投資経済を核にした 4v4 タクティカルシューターのプロジェクトです。  
このリポジトリには、MonoGame 製のプレイアブルプロトタイプと、今後の実装基準になる仕様書を置いています。

現在の公開版は `RYZECHo.Prototype v0.0.3` です。完成版 FPS ではなく、A/B 2 サイト構成のタクティカルスライスを先に触れるための Windows 向けプロトタイプとして扱います。

## 現在の実装物

現状の `RYZECHo.Prototype` は、フル 3D の完成版ではなく、コア体験を素早く触るためのトップダウン戦術スライスです。主に次の要素を先行検証しています。

- 初回構築フェーズと 8 種の設置物
- ラウンド前の 4 人ボス選択、4 人個別投資、ULT ポイント進行
- 4v4、A/B 2 サイト、ボム設置と解除、攻守交代、OT を含むラウンド進行
- 9 種のメイン武器と 2 種のサブウェポン、一覧型ショップ UI を含むロードアウト
- 100 度視界と `SR` 80 度視界、シェア・ビジョンを前提にした戦闘
- 波紋 UI、画面端インジケーター、味方共有を含む音の可視化
- 6 エージェントの切替と、戦闘中に使えるスキル / ULT のプロトタイプ
- AI のエージェント別スキル判断と、敵味方の簡易スキル発動
- ローカル保存のランク、XP、契約進行、見た目 inventory、非 P2W のコスメストア導線

未反映項目は [プロトタイプ差分メモ](docs/prototype-alignment.md) に整理しています。

## 技術構成

- `.NET 9`
- `MonoGame.Framework.DesktopGL`
- `FontStashSharp.MonoGame`
- C# と MonoGame プリミティブによる描画

### 再設計方針

現在の MonoGame プロトタイプは、プレイ可能状態を保ちながら段階的に責務分離しています。画面寸法、グリッド、試合ルール、経済、視野、固定ロスターは `Domain` 側の `GameLayout` / `GameRules` / `RosterCatalog` に集約し、`GameModel` はシミュレーション状態とフェーズ進行に寄せていきます。

## 起動

```powershell
dotnet run --project .\RYZECHo.Prototype\RYZECHo.Prototype.csproj
```

## 操作

### 構築

- `1 / 2 / 3 / 4 / 5`: 基本建築物選択
- `Tab`: 全建築物を順送り
- `6`: 使用エージェント切替
- `左クリック`: 強調スロットに設置
- `右クリック`: 設置物を撤去して AP を返還
- `Q / E`: 設置物スキン切替
- `R`: 会場広告テーマ切替
- `T`: 現在のコスメストア枠を購入 / 選択
- `Enter`: 構築確定

### ラウンド準備

- `1 / 2 / 3 / 4`: ボス選択
- `Q / E`: 選択中タブの装備変更
- `R`: メイン / サブウェポンタブ切替
- `A / D`: 選択中ボスの投資額調整
- `T`: 現在のコスメストア枠を購入 / 選択
- `Enter`: 出撃

### 戦闘

- `W / A / S / D`: 移動
- `マウス`: 照準
- `左クリック長押し`: 射撃
- `1 / 2 / 3`: エージェントのスキル 1 / スキル 2 / ULT
- `Q / E`: メイン / サブウェポン切替
- `F`: ボム設置 / 解除

### 共通

- `Space`: 説明オーバーレイ表示切替
- `R`: 勝敗後に再挑戦

## 既知の制限

- MonoGame のトップダウン戦術スライスで、完成版の 3D FPS や本番向けフロントエンドではありません
- エージェント 6 名のスキル / ULT と AI 使用判断はトップダウン向けの基礎実装で、完成版の Unity 演出や購入コストは未調整です
- マップは固定の A/B 2 サイト構成で、完全な本番マッププールやオンライン対戦は未実装です
- 構築レギュレーションは 3 レーンを保つ近似判定で、最終的なマップ審査ツールではありません
- プロフィール保存とコスメ inventory はローカル JSON ベースで、アカウント同期やサーバー権威の進行管理、本番課金は未実装です

詳細は [v0.0.3 リリースノート](docs/release-v0.0.3.md) と [プロトタイプ差分メモ](docs/prototype-alignment.md) を参照してください。

## Assets

- `RYZECHo.Prototype/Assets` 配下に `Audio / Characters / Environment / Weapons / UI / VFX / Fonts / Data` を整理済みです
- `tools/Generate-PlaceholderAssets.ps1` で仮アセットを再生成できます
- 本番アセットは同名差し替えか、各カテゴリへの追加で運用できます
